
# 🛒 Retail Mart Sales Analytics Dashboard

# Project Overview

This project is an interactive Power BI dashboard developed to analyze RetailMart sales performance. It provides business insights into sales, profit, customers, products, and regional performance using Power BI, DAX, and Power Query.

---

# Business Objectives

- Analyze Sales Performance
- Track Profitability
- Monitor Customer Behaviour
- Identify Top Performing Products
- Analyze State-wise Sales
- Compare Brand Performance

---

# Tools Used

- Power BI Desktop
- Power Query
- DAX
